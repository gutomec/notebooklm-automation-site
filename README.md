## O Problema

Você tem **fontes incríveis** — papers, vídeos, documentos, relatórios. Mas transformar isso em conteúdo publicável é uma maratona:

- ⏰ **10+ horas** para criar um podcast a partir de um paper
- 💸 **$500+** em locução profissional por episódio
- 🔄 **Trabalho repetitivo** de copiar, colar, reformatar, traduzir
- 😤 **Ferramentas fragmentadas** — uma pra slides, outra pra quiz, outra pra áudio

E se você pudesse automatizar **tudo isso** com um único sistema?

---

## A Solução

O **NotebookLM Automation Squad** é um time de **6 agentes de IA especializados** que trabalham juntos para:

1. **Ingerir** suas fontes (URLs, PDFs, vídeos do YouTube, Google Drive)
2. **Pesquisar** profundamente — não é resumo genérico, é análise com citações
3. **Gerar** 10 tipos de conteúdo profissional automaticamente
4. **Baixar** tudo organizado, pronto para publicar

```
Suas Fontes                         Seus Conteúdos
─────────────                       ──────────────
📄 Papers          ┌──────────┐     🎙️ Podcasts (50+ idiomas)
📹 Vídeos     ───▶ │  SQUAD   │ ──▶ 🎬 Vídeos
📑 PDFs            │ 6 agents │     📊 Slides (PPTX)
🔗 URLs            │ 35 tasks │     ❓ Quizzes
📝 Notas           └──────────┘     🃏 Flashcards
                                    📄 Relatórios
                                    🧠 Mind Maps
                                    📈 Infográficos
                                    📋 Data Tables
                                    🎥 Vídeos Cinemáticos
```

**⏱️ Tempo total: ~15 minutos.** De fontes brutas a 10 formatos publicáveis.
