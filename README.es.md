## El Problema

Tienes **fuentes increíbles** — papers, vídeos, documentos, informes. Pero convertirlos en contenido publicable es una maratón:

- ⏰ **10+ horas** para crear un podcast a partir de un paper
- 💸 **$500+** en locución profesional por episodio
- 🔄 **Trabajo repetitivo** — copiar, pegar, reformatear, traducir
- 😤 **Herramientas fragmentadas** — una para slides, otra para quizzes, otra para audio

¿Y si pudieras automatizar **todo eso** con un único sistema?

---

## La Solución

El **NotebookLM Automation Squad** es un equipo de **6 agentes de IA especializados** que trabajan juntos para:

1. **Ingerir** tus fuentes (URLs, PDFs, vídeos de YouTube, Google Drive)
2. **Investigar** a profundidad — no resúmenes genéricos, sino análisis con citas
3. **Generar** 10 tipos de contenido profesional automáticamente
4. **Descargar** todo organizado, listo para publicar

```
Tus Fuentes                         Tu Contenido
───────────                         ────────────
📄 Papers          ┌──────────┐     🎙️ Podcasts (50+ idiomas)
📹 Vídeos     ───▶ │  SQUAD   │ ──▶ 🎬 Vídeos
📑 PDFs            │ 6 agentes│     📊 Slides (PPTX)
🔗 URLs            │ 35 tareas│     ❓ Quizzes
📝 Notas           └──────────┘     🃏 Flashcards
                                    📄 Informes
                                    🧠 Mapas Mentales
                                    📈 Infografías
                                    📋 Tablas de Datos
                                    🎥 Vídeos Cinemáticos
```

**⏱️ Tiempo total: ~15 minutos.** De fuentes brutas a 10 formatos publicables.
